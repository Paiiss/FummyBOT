## How to get a bot token
* Open telegram and search for @BotFather
* Type / newbot
* Follow all the instructions given, and copy the token into the token column in ```setting.json```


## ✍️ Editing the file
Edit the required value in `config.json`.
```json
{
    "Token": "TOKEN BOT",
    "paisKey": "Fummy",
    "lolKey": "Apikey",
    "vhKey": "Apikey" 
}
```

# Requirements
* [Node.js](https://nodejs.org/en/)
* [Git](https://git-scm.com/downloads)
* Intention
* Any text editor

# Installation

## Cloning this repo
```cmd
> git clone https://github.com/Paiiss/FummyBOT.git
> cd FummyBOT
```

## Install the package
```cmd
> npm i
```

## Run the bot
```cmd
> npm start
```

## Note:
* You can use node / nodemon, you can set the settings in ```package.json```
